## 2. Journey: New User Onboarding

### Journey Overview

| Attribute | Value |
|-----------|-------|
| **Persona** | สมชาย |
| **Goal** | สมัครใช้งาน JECO Platform |
| **Entry Point** | QR Code จากโฆษณา |
| **Duration** | 3-5 นาที |
| **Success** | เข้าสู่ Dashboard พร้อมใช้งาน |

### Journey Steps

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         NEW USER ONBOARDING JOURNEY                         │
└─────────────────────────────────────────────────────────────────────────────┘

    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
    │  Scan   │───▶│ Splash  │───▶│  Login  │───▶│   OTP   │───▶│Dashboard│
    │   QR    │    │ Screen  │    │  Phone  │    │ Verify  │    │  Home   │
    └─────────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘
         │              │              │              │              │
    ─────┴──────────────┴──────────────┴──────────────┴──────────────┴─────

    Touchpoint:   Marketing     App          App          SMS          App
    Emotion:      Curious       Excited      Neutral      Waiting      Happy
    Duration:     -             2 sec        30 sec       30 sec       -
```

### Detailed Steps

#### Step 1: Discover & Access

| Element | Description |
|---------|-------------|
| **Trigger** | เห็นโฆษณา / QR Code ที่ร้าน Jaymart |
| **Action** | Scan QR หรือกดลิงก์ |
| **System** | Redirect ไปยัง JECO Web App |
| **Emotion** | Curiosity, Interest |

#### Step 2: Splash Screen (2 seconds)

| Element | Description |
|---------|-------------|
| **Display** | JECO Logo + Loading animation |
| **System** | Check device, Load initial config |
| **Duration** | 2 seconds max |
| **Next** | Auto-redirect to Login |

#### Step 3: Login with Phone

| Element | Description |
|---------|-------------|
| **Display** | Phone number input field |
| **Action** | กรอกเบอร์โทร 10 หลัก |
| **Validation** | Format: 0XX-XXX-XXXX |
| **Button** | "รับรหัส OTP" |
| **Emotion** | Neutral (familiar process) |

#### Step 4: OTP Verification

| Element | Description |
|---------|-------------|
| **Display** | 6-digit OTP input |
| **System** | ส่ง SMS ไปยังเบอร์ที่กรอก |
| **Timeout** | 60 seconds |
| **Retry** | กด "ส่งรหัสใหม่" หลัง timeout |
| **Success** | รหัสถูกต้อง → Create session |
| **Failure** | 3 ครั้ง → Block 15 นาที |

#### Step 5: Dashboard Home

| Element | Description |
|---------|-------------|
| **Display** | Welcome message + Main features |
| **Features** | Chat, Shop, Wallet, Loan, Profile |
| **First-time** | Optional onboarding tooltip tour |
| **Emotion** | Accomplished, Ready to explore |

### Pain Points & Solutions

| Pain Point | Solution |
|------------|----------|
| OTP ไม่เข้า | Resend option + Contact support link |
| Process ยาว | Minimal steps (2 screens only) |
| ไม่เข้าใจวิธีใช้ | Optional onboarding tour |
